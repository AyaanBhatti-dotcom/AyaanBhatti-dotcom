:: PROJECT 02 :: HONEYPOT RESEARCH ::  


(actual logs!!) 		ok this one is pretty cool and i am not gonna lie i feel like a real security researcher now lol. a honeypot is basically a fake server that u set up to LOOK like a real vulnerable system, and then when hackers try to break into it u can watch what they do and learn their techniques!!

i set mine up on a VPS running a fake SSH server (using cowrie honeypot software) and pointed a public IP at it and within like 1 minute i was already catching real hackers in my trap XD they tried every username and password combo u can imagine. i saw someone try "admin/admin" and "root/123456" which like... seriously dude.

honeypot software: cowrie SSH honeypot
OS: raspbian (debian based)
findings: mostly automated bots, tons of credential stuffing attempts
coolest thing: someone tried to download a botnet installer on my fake system lmao

> this is for my research project for my upgrading my knowledge
