:: PROJECT 02 :: HONEYPOT RESEARCH ::  


(actual logs!!) 		ok this one is pretty cool a honeypot is basically a fake server that u set up to LOOK like a real vulnerable system, and then when hackers try to break into it u can watch what they do and learn their techniques i used claude code to analyze the logs (on my github)

i set mine up on a VPS running a fake SSH server (using cowrie honeypot software) and pointed a public IP at it and within 1 minute i was already catching real hackers in my trap... they tried every username and password combo u can imagine. stay vigilant, stay secure.

honeypot software: cowrie SSH honeypot
OS: raspbian (debian based)
findings: mostly automated bots, tons of credential stuffing attempts
coolest thing: someone tried to download a botnet installer on my fake system lmao

> this is for my research project for my upgrading my knowledge
